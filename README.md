# 📊 Ecommerce Sales Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze e-commerce sales performance. This project provides business insights into sales, profit, customer segments, product performance, regional trends, and shipping analysis using dynamic visualizations and KPIs.

---

## 🚀 Project Overview

This dashboard helps businesses monitor key sales metrics and identify trends to support data-driven decision-making.

The dashboard includes:
- Sales Performance Analysis
- Profit Analysis
- Customer Segment Analysis
- Product Performance
- Regional Sales Insights
- Shipping Analysis
- Interactive Filters & Drill-down

---

## 🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
- DAX

---

## 📈 Key KPIs

- 💰 Year-to-Date Sales
- 💵 Year-to-Date Profit
- 📦 Year-to-Date Quantity Sold
- 📊 Profit Margin

---

## 📊 Dashboard Features

- Sales by Category
- Top 5 Products by Sales
- Bottom 5 Products by Sales
- Sales by Region
- Sales by Shipping Type
- Sales by State (Map Visualization)
- Monthly Sales Trend
- Interactive Segment Filter
