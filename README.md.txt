# Ecommerce Sales Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes e-commerce sales performance using interactive visualizations and KPIs. It helps users understand sales trends, profit, regional performance, product performance, and shipping insights.

## 🛠 Tools Used
- Power BI
- Excel
- SQL

## 📊 Dashboard Features
- YTD Sales
- YTD Profit
- YTD Quantity
- Profit Margin
- Sales by Category
- Top 5 Products
- Bottom 5 Products
- Sales by Region
- Sales by Shipping Type
- Sales by State (Map)
- Interactive Segment Filter
